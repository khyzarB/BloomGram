# BloomGram 🌸

A digital bouquet builder — pick flowers, arrange a bouquet, write a card, and share a unique link with a loved one.

## Tech Stack

- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** SQLite (via better-sqlite3)

## Setup

```bash
# Backend (Terminal 1)
cd backend
npm install
node index.js
# → http://localhost:3001

# Frontend (Terminal 2)
cd frontend
npm install
npm run dev
# → http://localhost:5173
```

## How It Works

1. **Pick Flowers** — choose from 12 flowers, each with a unique meaning
2. **Arrange** — see a live bouquet preview with your selections
3. **Write a Card** — pick a theme, write a heartfelt message
4. **Share** — get a unique link to send via WhatsApp, email, or copy
