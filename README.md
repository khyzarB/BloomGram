# BloomGram

A digital bouquet builder — pick illustrated SVG flowers, add lush greenery, arrange on a draggable canvas, write a themed card, and share a unique link.

## Tech Stack

- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** SQLite (via better-sqlite3)

## Setup

```bash
npm install
npm run dev
# → http://localhost:5173
```

## Features

- 10 hand-illustrated SVG flowers (Rose, Peony, Tulip, Daisy, Sunflower, Lily, Lotus, Orchid, Lavender, Cherry Blossom)
- 5 greenery types (Fern, Eucalyptus, Ivy, Baby's Breath, Leaf Spray)
- Drag-to-reposition bouquet canvas with auto-arrange
- 6 physical-feeling card themes (Vintage, Blush, Botanical, Elegant, Midnight, Sunrise)
- Animated reveal page for recipients
- Share via link, WhatsApp, or Email
